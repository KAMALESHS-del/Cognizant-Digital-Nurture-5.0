Insertion Sort

Insertion Sort is a simple sorting algorithm that builds the sorted array one element at a time.

It works like arranging playing cards in your hand:

You pick one card at a time.
Insert it into its correct position among the already sorted cards.


         "Algorithm"
Assume the first element is already sorted.
Pick the next element (key).
Compare it with the sorted part.
Shift larger elements one position to the right.
Insert the key into its correct position.
Repeat until the array is sorted.
